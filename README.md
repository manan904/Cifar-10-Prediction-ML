# Cifar-10-Prediction-ML
Prediction of CIFAR-10 dataset using Random Forest Classifier.

This project uses the CIFAR-10 dataset and applies machine learning algorithms to predict the labels correctly.
This repo consists of 2 iPython Notebooks one to run locally on a machine and the other one for running on Google Colab.
